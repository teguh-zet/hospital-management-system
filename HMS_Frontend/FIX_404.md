# Fix Error 404 - Langkah Lengkap

## Error yang Terjadi:
```
GET http://localhost:3000/ net::ERR_HTTP_RESPONSE_CODE_FAILURE 404 (Not Found)
```

## Solusi Step-by-Step:

### 1. **STOP Server (WAJIB)**
   Di terminal yang menjalankan `npm run dev`:
   - Tekan `Ctrl + C` untuk stop server
   - Pastikan server benar-benar berhenti

### 2. **Clear Semua Cache**
   ```powershell
   cd "D:\projek zeyt\hms\HMS_Frontend"
   
   # Hapus Vite cache
   Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue
   
   # Hapus dist folder jika ada
   Remove-Item -Recurse -Force dist -ErrorAction SilentlyContinue
   ```

### 3. **Clear Browser Cache (PENTING!)**
   
   **Opsi A - Hard Refresh:**
   - Tekan `Ctrl + Shift + R` (Windows)
   - Atau `Ctrl + F5`
   
   **Opsi B - Clear Cache Manual:**
   - Tekan `Ctrl + Shift + Delete`
   - Pilih "Cached images and files"
   - Pilih "All time"
   - Klik "Clear data"
   
   **Opsi C - Incognito/Private Window (RECOMMENDED):**
   - Buka Incognito/Private window
   - Akses `http://localhost:3000`
   - Ini akan bypass semua cache

### 4. **Restart Server**
   ```powershell
   cd "D:\projek zeyt\hms\HMS_Frontend"
   npm run dev
   ```

### 5. **Tunggu Server Ready**
   Pastikan muncul pesan:
   ```
   VITE v5.4.21  ready in xxx ms
   ➜  Local:   http://localhost:3000/
   ```

### 6. **Akses di Browser**
   - **JANGAN** ketik URL manual
   - **Klik** URL yang muncul di terminal: `http://localhost:3000/`
   - Atau copy-paste URL dari terminal
   - Atau gunakan Incognito window

### 7. **Cek Console Browser (F12)**
   - Buka Developer Tools (F12)
   - Tab Console
   - Harus muncul: "Vue app mounted successfully"
   - Harus muncul: "SimpleHome component mounted successfully"

## Jika Masih Error:

### Test 1: Cek Apakah Server Benar-benar Running
   ```powershell
   # Di terminal baru, test koneksi:
   curl http://localhost:3000
   # Atau buka di browser: http://localhost:3000
   ```

### Test 2: Cek Port
   Pastikan tidak ada aplikasi lain di port 3000:
   ```powershell
   netstat -ano | findstr :3000
   ```

### Test 3: Coba Port Lain
   Edit `vite.config.js`:
   ```js
   server: {
     port: 3001,  // Ganti ke 3001
   }
   ```
   Lalu akses: `http://localhost:3001`

### Test 4: Cek File index.html
   Pastikan file `public/index.html` ada dan berisi:
   ```html
   <div id="app">...</div>
   <script type="module" src="/src/main.js"></script>
   ```

## Troubleshooting Lanjutan:

### Jika Error "Cannot find module"
   ```powershell
   npm install
   ```

### Jika Error di Console tentang import
   - Cek apakah semua file yang diimport ada
   - Cek path import (harus relatif dari file)

### Jika Halaman Blank
   - Cek Console untuk error JavaScript
   - Cek Network tab untuk file yang gagal load
